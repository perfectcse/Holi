# 🌈 Holi Wishes App – React SPA

A colorful and interactive **React Single Page Application (SPA)** built using **React + Vite**.
This project delivers a festive experience with dynamic wishes, confetti animations, smooth UI transitions, and an embedded Holi music playlist.

---

## 🔗 Live Demo

👉 https://wishholi.netlify.app/

---

## 🚀 Features

* 🌈 Animated gradient background
* 🎉 Confetti effects on interaction
* 💬 Dynamic wish style toggle (Traditional / Fun / Professional)
* 🎶 Embedded Holi song playlist
* ✨ Smooth slide animations
* 📱 Fully responsive design
* 🌍 Deployed on Netlify

---

## 🛠️ Tech Stack

* ⚛️ React (Functional Components)
* 🪝 React Hooks (`useState`, `useEffect`)
* ⚡ Vite (Fast build tool)
* 🎨 CSS3 (Animations + Glassmorphism UI)
* 🎉 canvas-confetti (Animation library)
* 🌐 Netlify (Deployment)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone YOUR_GITHUB_REPO_LINK
cd holi-app
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### ▶️ Run in Development Mode

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```
